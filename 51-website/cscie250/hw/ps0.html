Instructions

The purpose of this assignment is to get you set up for the real assignments to come. You will set up an editing environment for Ocaml, make sure you can compile and run some toy programs, and figure out the assignment submission system.

Part 0: Signing up for a FAS account

If you haven't already done so, activate your Harvard FAS Computer Account by following the instructions at https://www.fas.harvard.edu/fasit/utilities/activate-pin/.

Part 1: Signing up for a bulletin board account

Go to forums.seas.harvard.edu/cs51, click Register (top left), and sign up for an account, using your FAS account name as your screenname. If you have problems with any of the rest of this assignment (or any of the other assignments), post to this bulletin board, or e-mail the TF at brian_young@post.harvard.edu.

Part 2: Installing Ocaml

  • Ocaml is already installed on the FAS nice servers. You can use it there (run ocaml), but we'd recommend also installing it on your own computer. To do so, download the appropriate package for your operating system from the Ocaml download page and then follow the directions.
  • To test that you can get Ocaml working on the FAS machines, use SSH to login to nice.fas.harvard.edu:
    • From a terminal, type ssh userid@nice.fas.harvard.edu, using the correct userid of course.
    • From a graphical ssh client, follow the directions :)
    Now run ocaml.
  • To test locally, just run ocaml from a terminal ("Command Prompt" in Windows). Whether you're running on the server or locally, you should now see the Ocaml prompt: 
    $ ocaml
        Objective Caml version 3.11.1

#

    If you type 

    "are we there yet?";;
    you should get the output: 
    - : string = "are we there yet?"

    Congratulations! If you got this far, Ocaml works. Type 

    #quit;;
    or Ctrl-D to exit Ocaml. Now you just have to learn how to write useful programs.
There are also instructions for getting OCaml running in Eclipse, which is a free IDE that will also work for java.

Part 3: Downloading the code and filling out the survey

Click here to download ps0.ml, and click here to download the Makefile. As above, you can work on nice.fas or locally on your computer. Save both files where you intend to do your work. To upload your downloaded files to nice, you'll need an FTP client (see instructions page, under "Uploading your work to nice").

Your job is to edit ps0.ml, setting all the variables to appropriate values.

If you are using Eclipse, you will have to create a new Ocaml Managed Project (see instructions above for using Eclipse) for PS0. Copy both files into the folder Eclipse creates for your new project. You can then edit ps0.ml.

Once you're done, make sure the file still compiles: if you're using Eclipse, use the instructions for that. If you're using emacs, use the provided Makefile by typing make in the appropriate directory: 

$ make
ocamlc -o survey ps0.ml

$ ./survey

Part 4: Submitting

To submit the assignment, first rename your ps0.ml to userid.ps0.ml, replacing userid with your FAS username. Then e-mail it to the TF at brian_young@post.harvard.edu before the due date.
